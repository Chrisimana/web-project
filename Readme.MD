# Web Project Collection 🚀

Sebuah koleksi proyek web interaktif yang dibangun menggunakan **HTML**, **CSS**, dan **JavaScript**. Repository ini berfungsi sebagai portfolio yang menampilkan berbagai aplikasi web dengan fokus pada fungsionalitas dan user experience.

---

## 📂 Struktur Repository

```
web-project/
├── index.html           # Halaman utama 
├── main.js              # Script utama untuk halaman portfolio
├── projects-data.js     # Data proyek yang ditampilkan
├── style.css            # Stylesheet untuk halaman utama
└── project/             # Folder berisi semua proyek
    ├── focusflip/
    ├── generator-quote/
    ├── jam-digital/
    ├── kalkulator/
    ├── kalkulator-bmi/
    ├── kalkulator-tip/
    ├── kalkulator-usia/
    ├── konversi-mata-uang/
    ├── noted/
    ├── pemeriksa-kekuatan-sandi/
    ├── pemutar-musik/
    ├── penggeser-gambar/
    ├── penghitung-waktu-mundur/
    ├── to-do-list/
    └── topwatch/
```

## 🎯 Daftar Proyek

| Proyek | Deskripsi | Fitur Utama |
|--------|-----------|-----------|
| **focusflip** | Aplikasi focus timer | Manajemen waktu fokus |

| **jam-digital** | Jam digital realtime | Display waktu digital |
| **kalkulator** | Kalkulator dasar | Operasi matematika dasar |
| **kalkulator-bmi** | Kalkulator Indeks Massa Tubuh | Hitung BMI |
| **kalkulator-tip** | Kalkulator tip/gratuity | Hitung persentase tip |
| **kalkulator-usia** | Kalkulator usia | Hitung usia dari tanggal lahir |
| **konversi-mata-uang** | Konverter mata uang | Konversi nilai tukar |
| **noted** | Aplikasi catatan | CRUD untuk catatan |
| **pemeriksa-kekuatan-sandi** | Password strength checker | Validasi kekuatan sandi |

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur dan markup
- **CSS3** - Styling dan layout
- **JavaScript** - Interaktivitas dan logika

---

## 📋 Struktur Setiap Proyek

Setiap proyek dalam folder `project/` memiliki struktur standar:

```
nama-project/
├── index.html     # File HTML utama
├── script.js      # Logic JavaScript
├── style.css      # Stylesheet proyek
├── Readme.MD      # Dokumentasi proyek
└── Screenshot/    # Folder preview
```

---

## 🚀 Cara Menggunakan

1. **Akses Portfolio Utama**
   - Buka `index.html` di browser untuk melihat daftar semua proyek
   
2. **Jalankan Proyek Individual**
   - Navigate ke folder proyek yang diinginkan
   - Buka `index.html` di browser
